# BrainLM Wiki

BrainLM is a decoder-only transformer for fMRI that learns from masked reconstruction. This wiki covers setup, architecture, data format, training, finetuning, and Hugging Face Hub usage.

## Pages

- **[Installation](Installation)** — Environment setup and dependencies
- **[Architecture](Architecture)** — Decoder-only transformer, block-causal attention, masking
- **[Data Format](Data-Format)** — Arrow dataset schema and coordinate format
- **[Training](Training)** — Pretraining from scratch, hyperparameters, resume
- **[Finetuning](Finetuning)** — Loading a pretrained model and adding an MLP head
- **[HuggingFace Hub](HuggingFace-Hub)** — Loading and pushing models
